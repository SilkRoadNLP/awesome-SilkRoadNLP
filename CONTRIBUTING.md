# Contributing to SilkRoadNLP

Thank you for your interest in contributing to the SilkRoadNLP community! Every contribution — no matter how small — helps close the resource gap for Iranian languages.

---

## Ways to Contribute

### 1. Add a Resource to the Awesome List

If you know of a paper, dataset, tool, or model related to NLP for any Iranian language, add it to [awesome-SilkRoadNLP](https://github.com/SilkRoadNLP/awesome-SilkRoadNLP).

**Steps:**

1. **Fork** the [awesome-SilkRoadNLP](https://github.com/SilkRoadNLP/awesome-SilkRoadNLP) repository
2. Find the appropriate language page under `languages/` (e.g., `languages/persian.md`, `languages/kurdish.md`)
3. Add your entry to the relevant section using this format:

```markdown
| [Paper Title](https://link-to-paper) | Author1, Author2, Author3 | Year | [![GitHub](https://img.shields.io/badge/GitHub-Code-black?logo=github)](https://github.com/...) [![HuggingFace](https://img.shields.io/badge/🤗-Model-yellow)](https://huggingface.co/...) |
```

4. If no section fits, create a new one with a clear heading
5. Submit a **Pull Request** with a brief description of what you added

**What to include for each entry:**
- Paper title with a link (ACL Anthology, arXiv, or publisher DOI preferred)
- Full author list
- Publication year
- GitHub link (if code/data is available)
- HuggingFace link (if models/datasets are hosted there)

### 2. Share Your Project

Working on NLP tools or datasets for an Iranian language? You can:

- **Add it to the awesome list** via a PR (see above)
- **Fork it under the SilkRoadNLP org** — if your project aligns with the community's mission, reach out and we can host a fork or mirror under the [SilkRoadNLP GitHub organization](https://github.com/SilkRoadNLP) for greater visibility
- **Present it at a SilkRoadNLP workshop** — submit your work to future SilkRoadNLP calls for papers

### 3. Contribute Data for Under-resourced Languages

Many Iranian languages have **zero** computational resources. Even basic contributions are valuable:

- Raw or transcribed text corpora
- Parallel translations (even small-scale)
- Word lists or dictionaries
- Annotated datasets (NER, POS, sentiment, etc.)
- Evaluation benchmarks

Languages that need the most help: **Balochi**, **Luri**, **Ossetic**, **Gilaki**, **Mazandarani**, **Shughni**, **Gorani**, and many more.

### 4. Improve Existing Pages

- Fix broken links or outdated information
- Add missing GitHub/HuggingFace links for existing entries
- Improve descriptions or add context
- Fix typos or formatting issues

### 5. Suggest a New Language Page

If a language isn't covered yet, open an [issue](https://github.com/SilkRoadNLP/awesome-SilkRoadNLP/issues) or submit a PR adding a new page under `languages/`. Use the existing pages as a template.

---

## Guidelines

- **Relevance** — Every entry must be directly related to NLP for at least one Iranian language
- **Accuracy** — Verify paper titles, author names, and links before submitting
- **No duplicates** — Check that the resource isn't already listed
- **Proper attribution** — Always credit the original authors
- **Language pages** — Add entries to the most specific language page; cross-list in `languages/multilingual.md` if a resource covers multiple languages

---

## Pull Request Process

1. Fork the repository and create a branch (`feature/add-xyz-paper`)
2. Make your changes
3. Verify all links work
4. Submit the PR with a short description of what was added
5. A maintainer will review and merge

---

## Code of Conduct

Be respectful, inclusive, and constructive. This community spans many countries, cultures, and languages — that diversity is our strength.
